# Number Guessing Game 🎯

A command-line Python game where the computer picks a random number and you try to guess it.

Built as my first Python project while learning programming.

## How to play

1. Choose a difficulty level (Easy / Medium / Hard)
2. The computer picks a secret number in that range
3. Enter your guesses — the game tells you if you're too high or too low
4. Hot/cold hints help you get closer
5. Guess the number before you run out of attempts!

## Difficulty levels

| Level  | Range  | Attempts |
|--------|--------|----------|
| Easy   | 1–50   | 10       |
| Medium | 1–100  | 7        |
| Hard   | 1–200  | 5        |

## Run it

Make sure Python is installed, then:

```
python number_guessing_game.py
```
